# Introduction to Python Programming: with Claude
## Compact Tik TOC Architecture

**Book type:** Course textbook / beginner programming guide  
**Status:** Retrospective architecture for a developed manuscript  

## Core Thesis

This book teaches Python programming in the age of AI by helping beginners use Claude as a tutor, debugger, and explainer without skipping the mental work that builds programming ability.

## Primary Reader

A first-time or early Python learner, possibly in a college course, bootcamp, or self-study setting, who wants practical coding skill and responsible AI support.

## Learning Arc

1. **Programming basics:** variables, types, control flow, functions, collections, and files.
2. **Problem solving:** decomposition, debugging, testing, data handling, and libraries.
3. **AI-assisted practice:** asking for explanations, comparing solutions, debugging with evidence, and building small projects.

## Chapter Architecture

Each chapter should include a concept, a worked example, a practice set, a debugging case, and an AI-use boundary: what Claude may explain versus what the learner must attempt.

## Terminal Capability

The reader can write, run, debug, and explain small Python programs while using Claude to support learning rather than replace it.

## Adoption Risks

- Claude may make examples too easy and reduce practice.
- Version and environment details may age.
- The book must keep unassisted coding checks visible.

## Production Notes

Use Tik TOC to enforce a project spiral: every new feature should reappear in a larger program.
